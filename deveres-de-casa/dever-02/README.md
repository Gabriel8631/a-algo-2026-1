
 Algoritmo Fatorial Recursivo

Este projeto implementa o cálculo do fatorial de um número utilizando recursão em Python.

 Funcionamento

O algoritmo utiliza a definição matemática do fatorial:

n! = n × (n-1)!

Com caso base:
0! = 1

 Complexidade

- Tempo: O(n)
- Espaço: O(n)

A função realiza n chamadas recursivas, cada uma executando uma operação constante.

 Testes de desempenho

O tempo de execução foi medido para os seguintes valores:

- n = 10
- n = 100
- n = 500
- n = 1000

Observamos que o crescimento linear do tempo de execução, consistente com a complexidade O(n).
